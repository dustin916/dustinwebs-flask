# DustinWebs

DustinWebs was designed as a professional business landing page for my freelance web development services. Moving way from the hobbyist feel of my earlier projects, this site emphasized better design, solid code structure, and strong user input validation.

## Tech Stack
- Backend: Python (Flask) using Blueprints for organized routing.
- Forms & Validation: Flask-WTF and WTForms (Implementing server-side validation and CSRF protection).
- Frontend: HTML5, CSS3, Jinja2 Templates
- Styling: Bootstrap 5.3.0 for layout, supplemented by custom CSS.
Automation: smtplib for two-way communication (Auto-responding to potential clients while alerting me of new leads).

## Key Features
- Validated Lead Generation: Built a sophisticated contact system using Flask-WTF. This ensured that all potential client inquiries were validated for correct email formatting and required data before processing.
- Modern Business Aesthetic: Utilized Bootstrap 5.3 to create a responsive, professional UI that emphasized clarity, services offered, and past project success.
- Service-Oriented Architecture: Designed specific routes and templates for "Services" and "Portfolio," creating a clear marketing funnel for prospective clients.
- Automated Workflow: The backend handles lead capture by sending a confirmation receipt to the user and a detailed notification to my professional email, ensuring zero lead latency.

## What I Learned
- Security & Validation: This project marked my transition to using professional libraries like Flask-WTF, teaching me the importance of filtering and validating user input to prevent common web vulnerabilities.
- Scalable Design: By implementing Flask Blueprints, I learned how to structure an application that can easily scale as more services or pages are added.
- Professional Branding: I focused heavily on UI/UX principles here, ensuring the typography, spacing, and layout reflected a business-ready environment.

## Status
- Project mostly completed - few pieces of information missing, but functionality is done; website was never deployed.
